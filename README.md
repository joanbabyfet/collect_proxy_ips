## About
采集免费proxy ip，制作自己的代理ip池

## Feature

* 采集proxy ip, 来源为 https://free-proxy-list.net
* 可制作自己的代理ip池或爬虫用ip

## Requires
Python 3.11.0  
requests 2.28.1  
BeautifulSoup 4.11.1   

## Usage
```
python main.py
```
